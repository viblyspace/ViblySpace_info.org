What is the Internet of Things (IoT)?

The Internet of Things (IoT) refers to the network of physical objects—"things"—that are embedded with sensors, software, and other technologies to connect and exchange data with other devices and systems over the internet.

Simple Definition:

IoT is like giving everyday objects a brain and internet connection. Just like people use the internet to communicate, IoT devices use the internet to share information and work together.

How IoT Works: The Basic Process

IoT systems follow a simple pattern: collect data, send it over the internet, process it, and take action.

Sensors
Gateway
Cloud
User

The 4-Step IoT Process

1. Data Collection

Sensors gather information from the physical world (temperature, motion, light, etc.)

2. Data Transmission

Data is sent to the cloud via WiFi, Bluetooth, cellular, or other networks

3. Data Processing

Software analyzes the data and decides what action to take

4. Action/Response

The system responds by sending commands back to devices

Arduino
Open-source electronics platform for beginners and professionals

Arduino is an open-source electronics platform based on easy-to-use hardware and software. It's ideal for beginners in IoT and electronics due to its simple programming environment and extensive community support.

Arduino Board Overview

Arduino Uno R3 Board Diagram

Key Specifications

Input Voltage

Pin Configuration

Power Pins

  • 5V - Regulated 5V output
  • 3.3V - Regulated 3.3V output
  • GND - Ground pins
  • Vin - Input voltage

Digital I/O Pins

  • Pins 0-13 - Digital input/output

Other Pins

  • Reset - Reset the microcontroller
  • Pin 13 - Built-in LED

Programming Arduino

Arduino programs are called "sketches" and are written in a simplified version of C++ using the Arduino IDE.

Basic Structure of an Arduino Sketch

// Arduino sketch basic structure // This code runs once when the board starts void setup() {   // Initialize pins, serial communication, etc.   pinMode(13, OUTPUT); // Set pin 13 as output   Serial.begin(9600); // Start serial communication } // This code runs repeatedly after setup() void loop() {   // Main program logic goes here   digitalWrite(13, HIGH); // Turn on LED   delay(1000); // Wait for 1 second   digitalWrite(13, LOW); // Turn off LED   delay(1000); // Wait for 1 second }

Example: Reading Temperature Sensor

// Reading temperature from LM35 sensor const int tempPin = A0; // Temperature sensor connected to A0 void setup() {   Serial.begin(9600); } void loop() {   // Read analog value (0-1023)   int sensorValue = analogRead(tempPin);      // Convert to voltage (0-5V)   float voltage = sensorValue * (5.0 / 1023.0);      // Convert voltage to temperature (LM35: 10mV per degree C)   float temperature = voltage * 100;      // Print temperature to serial monitor   Serial.print("Temperature: ");   Serial.print(temperature);   Serial.println(" C");      delay(2000); // Wait 2 seconds before next reading }

Raspberry Pi
Small single-board computer for advanced IoT applications

Raspberry Pi is a series of small single-board computers developed to promote teaching of basic computer science in schools and developing countries. It's more powerful than Arduino and can run a full operating system, making it suitable for complex IoT applications.

  • 5V - 5V power
  • GND - Ground pins

  • GPIO 2-27 - Digital input/output
  • GPIO 12, 13, 18, 19 - Hardware PWM
  • GPIO 14, 15 - UART (serial)

Special Function Pins

  • GPIO 10, 11 - SPI
  • GPIO 3, 5 - I2C
  • GPIO 28-29 - Reserved for camera

Other Pins

  • ID_SD, ID_SC - EEPROM identification
  • GPIO 9, 11 - Reserved for display

Programming Raspberry Pi

Raspberry Pi supports multiple programming languages including Python, C/C++, JavaScript, and more. Python is the most popular choice for IoT applications.

Python GPIO Control Example

# Python script to control GPIO on Raspberry Pi import RPi.GPIO as GPIO import time # Set up GPIO mode GPIO.setmode(GPIO.BCM) # Disable warnings GPIO.setwarnings(False) # Define pin numbers led_pin = 18 button_pin = 17 # Setup pins GPIO.setup(led_pin, GPIO.OUT) GPIO.setup(button_pin, GPIO.IN, pull_up_down=GPIO.PUD_UP) # Function to blink LED def blink_led(times, speed):   for i in range(times):     GPIO.output(led_pin, GPIO.HIGH)     time.sleep(speed)     GPIO.output(led_pin, GPIO.LOW)     time.sleep(speed) try:   while True:     # Check if button is pressed     button_state = GPIO.input(button_pin)     if button_state == False:       print("Button pressed! Blinking LED...")       blink_led(5, 0.2)       time.sleep(0.2) # Debounce delay     else:       time.sleep(0.1) except KeyboardInterrupt:   print("Program stopped by user") finally:   GPIO.cleanup() # Clean up GPIO on exit

# Raspberry Pi web server to control LED from flask import Flask, render_template, request, jsonify import RPi.GPIO as GPIO app = Flask(__name__) # GPIO setup GPIO.setmode(GPIO.BCM) led_pins = {   'red': 17,   'green': 18,   'blue': 27 } # Initialize all LEDs to OFF for pin in led_pins.values():   GPIO.setup(pin, GPIO.OUT)   GPIO.output(pin, GPIO.LOW) @app.route('/') def index():   return render_template('index.html') @app.route('/led/<color>/<state>') def led_control(color, state):   if color in led_pins:     pin = led_pins[color]     if state == 'on':       GPIO.output(pin, GPIO.HIGH)       return jsonify({'status': 'success', 'message': f'{color} LED turned ON'})     elif state == 'off':       GPIO.output(pin, GPIO.LOW)       return jsonify({'status': 'success', 'message': f'{color} LED turned OFF'})   else:     return jsonify({'status': 'error', 'message': 'Invalid color'}) if __name__ == '__main__':   app.run(host='0.0.0.0', port=5000, debug=True)

Complete Arduino Code

This Arduino code reads data from all sensors and displays it on the LCD screen.

// Smart Plant Monitoring System // Reads soil moisture, temperature, humidity, and light level // Displays data on 16x2 LCD screen // Include necessary libraries #include <LiquidCrystal.h> // Library for LCD control #include <DHT.h> // Library for DHT11 sensor // Define sensor pins and constants #define SOIL_MOISTURE_PIN A0 // Analog pin for soil sensor #define LDR_PIN A1 // Analog pin for light sensor #define DHT_PIN 7 // Digital pin for DHT11 #define DHT_TYPE DHT11 // Define sensor type // Initialize LCD with pin connections // Parameters: RS, E, D4, D5, D6, D7 LiquidCrystal lcd(12, 11, 5, 4, 3, 2); // Initialize DHT sensor DHT dht(DHT_PIN, DHT_TYPE); // Setup function - runs once at startup void setup() { // Initialize serial communication for debugging Serial.begin(9600); // Initialize LCD with 16 columns and 2 rows lcd.begin(16, 2); // Start DHT sensor dht.begin(); // Print welcome message on LCD lcd.print("Plant Monitor"); lcd.setCursor(0, 1); // Move to second row lcd.print("Initializing..."); // Wait 2 seconds for sensors to stabilize delay(2000); // Clear LCD for data display lcd.clear(); } // Main loop - runs repeatedly void loop() { // Read soil moisture (0-1023) int soilValue = analogRead(SOIL_MOISTURE_PIN); // Convert to percentage (adjust these values based on your sensor) int soilMoisture = map(soilValue, 0, 1023, 0, 100); // Read temperature and humidity from DHT11 float temperature = dht.readTemperature(); // Read temperature in Celsius float humidity = dht.readHumidity(); // Read humidity percentage // Read light level from LDR (0-1023) int lightValue = analogRead(LDR_PIN); // Convert light value to percentage int lightLevel = map(lightValue, 0, 1023, 0, 100); // Display data on LCD lcd.setCursor(0, 0); // First row lcd.print("M:"); lcd.print(soilMoisture); lcd.print("% T:"); lcd.print((int)temperature); // Convert to integer for display lcd.print("C"); lcd.setCursor(0, 1); // Second row lcd.print("H:"); lcd.print((int)humidity); // Convert to integer for display lcd.print("% L:"); lcd.print(lightLevel); lcd.print("%"); // Also print to serial monitor for debugging Serial.print("Soil: "); Serial.print(soilMoisture); Serial.print("%, Temp: "); Serial.print(temperature); Serial.print("C, Humidity: "); Serial.print(humidity); Serial.print("%, Light: "); Serial.print(lightLevel); Serial.println("%"); // Wait 2 seconds before next reading delay(2000); }

Step by Step Instructions

Follow these steps carefully to build your Smart Plant Monitoring System.

1
Gather Components

Collect all the components listed in the Components section. Make sure you have everything before starting.

Checklist
  • Arduino Uno board
  • USB cable for programming
  • 16x2 LCD display
  • Soil moisture sensor
  • DHT11 temperature/humidity sensor
  • LDR (light sensor)
  • 10kΩ resistor (for LDR)
  • Potentiometer (for LCD contrast)
  • Breadboard and jumper wires
2
Set Up the LCD Display

Connect the LCD to the Arduino following the pin connections in the Circuit Diagram section.

Pro Tip

Use the potentiometer to adjust the LCD contrast. Turn it until you can clearly see the characters.

3
Connect the Sensors

Connect each sensor to the Arduino according to the pin assignments:

  • Soil moisture sensor to A0
  • DHT11 to digital pin 7
  • LDR to A1 (with 10kΩ resistor to GND)
4
Install Required Libraries

In the Arduino IDE, install these libraries:

  • LiquidCrystal (usually included by default)
  • DHT sensor library by Adafruit
How to Install Libraries

Go to Sketch > Include Library > Manage Libraries, then search for and install "DHT sensor library".

5
Upload the Code

Copy the complete code from the Code Explanation section and upload it to your Arduino.

Troubleshooting

If you get upload errors, check that you've selected the correct board (Arduino Uno) and port in the Tools menu.

6
Test the System

After uploading, you should see sensor readings on the LCD. Test each sensor:

  • Touch the soil sensor - moisture reading should change
  • Breathe on the DHT11 - temperature/humidity should change
  • Cover the LDR - light reading should decrease
7
Calibrate Sensors (Optional)

For more accurate readings, you can calibrate the sensors:

  • Soil sensor: Take readings in dry soil and water, then adjust the map() function
  • LDR: Measure in dark and bright light, adjust the map() function
