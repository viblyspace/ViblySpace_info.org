File Handling - Saving Data Permanently

Simple Explanation

File handling is like saving your work in a notebook instead of writing on a whiteboard. When you write on a whiteboard, it gets erased when you turn it off. But when you write in a notebook, your work stays there forever!

In programming, we use files to save data so it doesn't disappear when we close our program.

Reading from Files

# Reading a file is like reading a book
# Step 1: Open the file (like opening a book)
file = open("my_notes.txt", "r") # "r" means read

# Step 2: Read the content (like reading pages)
content = file.read()
print("File content:")
print(content)

# Step 3: Close the file (like closing the book)
file.close()

Writing to Files

# Writing to a file is like writing in a diary
# Step 1: Open the file for writing
file = open("my_diary.txt", "w") # "w" means write

# Step 2: Write some text
file.write("Dear Diary,\n")
file.write("Today I learned file handling in Python!\n")
file.write("It's like having a permanent memory for my programs.")

# Step 3: Close the file
file.close()

print("✅ Diary entry saved successfully!")

Web Programming Basics

Simple Explanation

Web programming is like building a house that people can visit through the internet.

HTML is the structure (walls, rooms), CSS is the paint and decoration, and Python (Flask) is the electricity and plumbing that makes everything work!

Creating a Simple Website with Flask

# Flask is like a magic box that turns Python into websites
# First, we need to install Flask: pip install flask

from flask import Flask

# Create our website app
app = Flask("MyWebsite")

# Create a home page (like the main door of our house)
@app.route("/")
def home():
    return """
<h1>Welcome to My Python Website!</h1>
<p>This is my first web page with Python.</p>
<a href="/about">Learn about me</a>
"""

# Create an about page (like another room in our house)
@app.route("/about")
def about():
    return """
<h1>About Me</h1>
<p>I'm a Grade 10 student learning web programming!</p>
<a href="/">Go back home</a>
"""

# Start the website
if __name__ == "__main__":
    app.run(debug=True)

How It Works

When you run this code, Flask creates a local website. You can visit it in your browser by going to http://localhost:5000

• The @app.route("/") tells Flask: "When someone visits the main page, show them this content"

• The @app.route("/about") tells Flask: "When someone visits /about, show them different content"

Data Analysis Introduction

Simple Explanation

Data analysis is like being a detective who looks at clues (data) to find patterns and solve mysteries.

For example, if you have test scores of all students, you can analyze them to find:

  • Who got the highest score?
  • What's the average score?
  • How many students passed?

Basic Data Analysis with Lists

# Let's analyze test scores of our class
scores = [85, 92, 78, 96, 88, 74, 91, 89]

# Basic analysis
average = sum(scores) / len(scores)
highest = max(scores)
lowest = min(scores)

print("📊 Test Score Analysis")
print(f"Number of students: {len(scores)}")
print(f"Average score: {average:.1f}")
print(f"Highest score: {highest}")
print(f"Lowest score: {lowest}")

# Find how many students passed (score >= 75)
passed = [score for score in scores if score >= 75]
print(f"Students passed: {len(passed)}/{len(scores)}")

# Sort scores to see ranking
sorted_scores = sorted(scores, reverse=True)
print(f"Top 3 scores: {sorted_scores[:3]}")

Error Handling

Simple Explanation

Error handling is like having a backup plan when something goes wrong. Instead of your program crashing, you can tell it what to do when it encounters problems.

Think of it like having an umbrella when it might rain - you're prepared for unexpected situations!

Basic Error Handling with Try-Except

# Without error handling - program crashes if something goes wrong
print("Let's divide two numbers")
num1 = int(input("Enter first number: "))
num2 = int(input("Enter second number: "))

result = num1 / num2 # This will crash if num2 is 0!
print(f"Result: {result}")

# With error handling - program continues gracefully
try:
    num1 = int(input("Enter first number: "))
    num2 = int(input("Enter second number: "))
    result = num1 / num2
    print(f"Result: {result}")
except ZeroDivisionError:
    print("❌ Error: Cannot divide by zero!")
except ValueError:
    print("❌ Error: Please enter valid numbers!")
except Exception as e:
    print(f"❌ An unexpected error occurred: {e}")
finally:
    print("✅ Program execution completed")

Common Error Types

  • ZeroDivisionError: When you try to divide by zero
  • ValueError: When you provide invalid input (like letters instead of numbers)
  • FileNotFoundError: When you try to open a file that doesn't exist
  • TypeError: When you use wrong data types (like adding string to number)
  • IndexError: When you try to access list items that don't exist

Practical Example: Safe File Reading

# Safe way to read files with error handling
def read_file_safely(filename):
    try:
        with open(filename, "r") as file:
            content = file.read()
            print(f"✅ Successfully read {filename}")
            return content
    except FileNotFoundError:
        print(f"❌ File '{filename}' not found!")
        return None
    except PermissionError:
        print(f"❌ No permission to read '{filename}'")
        return None
    except Exception as e:
        print(f"❌ Unexpected error: {e}")
        return None

# Test the function
content = read_file_safely("my_notes.txt")
if content:
    print("File content:")
    print(content)
else:
    print("Could not read the file.")

Final Project: Personal Finance Tracker

Create a complete application to track your income and expenses using everything you've learned!

# Personal Finance Tracker - Complete Project
class FinanceTracker:
    def __init__(self):
        self.transactions = []
        self.balance = 0
        self.load_data()

    def add_income(self, amount, description):
        self.balance += amount
        transaction = {
            "type": "Income",
            "amount": amount,
            "description": description
        }
        self.transactions.append(transaction)
        print(f"✅ Added income: ₹{amount} - {description}")
        self.save_data()

    def add_expense(self, amount, description):
        if amount > self.balance:
            print("❌ Not enough balance!")
            return
        self.balance -= amount
        transaction = {
            "type": "Expense",
            "amount": amount,
            "description": description
        }
        self.transactions.append(transaction)
        print(f"✅ Added expense: ₹{amount} - {description}")
        self.save_data()

    def show_summary(self):
        total_income = sum(t["amount"] for t in self.transactions if t["type"] == "Income")
        total_expense = sum(t["amount"] for t in self.transactions if t["type"] == "Expense")
        
        print("\n💰 FINANCIAL SUMMARY")
        print("=" * 30)
        print(f"Current Balance: ₹{self.balance}")
        print(f"Total Income: ₹{total_income}")
        print(f"Total Expenses: ₹{total_expense}")
        print(f"Number of Transactions: {len(self.transactions)}")

    def show_transactions(self):
        if not self.transactions:
            print("No transactions yet.")
            return
        print("\n📝 TRANSACTION HISTORY")
        for i, transaction in enumerate(self.transactions, 1):
            sign = "+" if transaction["type"] == "Income" else "-"
            print(f"{i}. {transaction['type']}: {sign}₹{transaction['amount']} - {transaction['description']}")

    def save_data(self):
        with open("finance_data.txt", "w") as file:
            file.write(f"Balance:{self.balance}\n")
            for transaction in self.transactions:
                file.write(f"{transaction['type']},{transaction['amount']},{transaction['description']}\n")

    def load_data(self):
        try:
            with open("finance_data.txt", "r") as file:
                lines = file.readlines()
                if lines:
                    self.balance = float(lines[0].split(":")[1])
                    for line in lines[1:]:
                        parts = line.strip().split(",")
                        if len(parts) == 3:
                            transaction = {
                                "type": parts[0],
                                "amount": float(parts[1]),
                                "description": parts[2]
                            }
                            self.transactions.append(transaction)
    except FileNotFoundError:
            # First time running - no data file yet
            pass

# Main program
tracker = FinanceTracker()

print("💼 PERSONAL FINANCE TRACKER")
print("1. Add Income")
print("2. Add Expense")
print("3. Show Summary")
print("4. Show Transactions")
print("5. Exit")

# Add some sample data
tracker.add_income(1000, "Pocket money")
tracker.add_income(500, "Birthday gift")
tracker.add_expense(200, "Lunch")
tracker.add_expense(150, "Books")

tracker.show_summary()
tracker.show_transactions()
