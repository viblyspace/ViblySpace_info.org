Module 1: HTML Fundamentals

What is HTML?

HTML (HyperText Markup Language) is the foundation of every website. It provides the structural framework that defines the content and layout of web pages.

HTML uses elements represented by tags to mark up different parts of content. These tags tell web browsers how to display the content.

Did You Know?

The first version of HTML was created by Tim Berners-Lee in 1991. He was trying to create a system for scientists to share research documents easily. Today, HTML is used by billions of websites worldwide!

Essential HTML Tags

<html> - The root element of an HTML page
<head> - Contains meta information about the document
<title> - Specifies a title for the document
<body> - Contains the visible page content
<h1> to <h6> - Defines HTML headings
<p> - Defines a paragraph
<a> - Defines a hyperlink
<img> - Defines an image

HTML Document Structure

Every HTML document follows a standard structure:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
  </body>
</html>

Module 2: CSS Styling

What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of documents written in HTML. While HTML defines the structure, CSS controls the visual appearance.

CSS allows you to apply styles such as colors, fonts, spacing, and layout to HTML elements, creating visually appealing web pages.

CSS Color Systems

Hex Codes: Six-digit codes representing colors (e.g., #FF0000 for red)

RGB: Defines colors using Red, Green, and Blue values (e.g., rgb(255, 0, 0))

Color Names: Predefined color names that browsers recognize (e.g., "red", "blue")

Common CSS Properties

color - Sets the text color
background-color - Sets the background color
font-family - Specifies the font for text
font-size - Sets the size of the font
margin - Sets the space outside an element
padding - Sets the space inside an element
border - Sets the border around an element

CSS Syntax

CSS rules consist of a selector and a declaration block:

selector {
  property: value;
}

Example:
h1 {
  color: blue;
  font-size: 24px;
  font-family: Arial, sans-serif;
}

Module 3: Web Elements

Integrating HTML and CSS

Effective web development requires the seamless integration of HTML and CSS. HTML provides the structural foundation, while CSS enhances it with visual design.

This module focuses on applying CSS to HTML elements to control their appearance, size, and positioning on the page.

CSS Sizing Properties

width - Sets the width of an element
height - Sets the height of an element
max-width - Sets the maximum width
min-height - Sets the minimum height
font-size - Controls text size

CSS Units of Measurement

px - Pixels (absolute measurement)

% - Percentage (relative to parent element)

em - Relative to the font-size of the element

rem - Relative to the font-size of the root element

vh/vw - Percentage of viewport height/width

Creating a Styled Element

HTML:
<div class="content-box">
  Welcome to our website!
</div>

CSS:
.content-box {
  width: 300px;
  height: 200px;
  background-color: #f0f8ff;
  border: 2px solid #4682b4;
  padding: 20px;
  margin: 15px;
  border-radius: 8px;
}

Module 4: Web Layout

Complete Web Page Development

Creating effective web pages requires understanding how to structure content and apply consistent styling across all elements.

This module covers the complete web development workflow, from planning to implementation, focusing on creating well-structured, visually cohesive web pages.

CSS Layout Methods

Normal Flow - The default browser layout

Flexbox - One-dimensional layout system

Grid - Two-dimensional layout system

Positioning - Precise element placement

Responsive Design Principles

Modern websites must provide optimal viewing experiences across various devices:

  • Fluid layouts that adapt to screen size
  • Media queries for device-specific styling
  • Flexible images and media
  • Readable typography at all sizes

Web Development Workflow

1. Planning and Design - Define the purpose, audience, and content structure

2. HTML Structure - Build the semantic HTML foundation

3. CSS Styling - Apply visual styles for usability and appeal

4. Testing and Refinement - Test across different browsers and devices

Module 5: Adding Images and Links

Making Your Website Interactive

Now that you know how to create text and style it, let's learn how to add images and links to make your website more interesting and interactive!

Images help tell stories and make your website visually appealing, while links let visitors navigate between pages.

Adding Images with HTML

The <img> tag is used to add images to your webpage:

<img src="cat.jpg" alt="A cute cat">

src tells the browser where to find the image file

alt provides a description for screen readers

Creating Links with HTML

The <a> tag creates clickable links:

<a href="https://www.google.com">Visit Google</a>

href tells the browser where the link should go

The text between the tags is what users see and click on

Module 6: Creating Lists and Tables

Organizing Information

Sometimes we need to present information in an organized way. HTML gives us special tags to create lists and tables that help visitors understand information quickly.

Think about how you might list your favorite games or create a schedule for your week - lists and tables are perfect for this!

HTML List Types

Ordered Lists (numbered):

<ol>
  <li>First item</li>
  <li>Second item</li>
</ol>

Unordered Lists (bulleted):

<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>

Creating Tables

Tables organize information into rows and columns:

<table>
  <tr>
    <th>Day</th>
    <th>Activity</th>
  </tr>
  <tr>
    <td>Monday</td>
    <td>Soccer practice</td>
  </tr>
</table>

Module 7: Fun with Colors and Fonts

Expressing Yourself with Design

Colors and fonts are like the personality of your website! They help create moods and make your content more enjoyable to read.

Just like you might choose different colors for different subjects in your notebook, you can choose colors and fonts that match what your website is about.

Color Your World

CSS lets you color almost anything on your webpage:

/* Text color */
color: red;

/* Background color */
background-color: blue;

/* Border color */
border-color: green;

You can use color names like "red" or "blue", or use special codes like #FF0000 for red.

Playing with Fonts

Change how your text looks with these CSS properties:

/* Change font family */
font-family: Arial, sans-serif;

/* Change font size */
font-size: 20px;

/* Make text bold */
font-weight: bold;

/* Make text italic */
font-style: italic;

Module 8: Your First Complete Website

Putting It All Together

Congratulations! You've learned the basic building blocks of web design. Now let's combine everything you've learned to create your first complete website.

We'll create a simple personal website that you can customize with information about yourself, your hobbies, and things you like!

Build Your Personal Website

<!DOCTYPE html>
<html>
<head>
  <title>My Personal Website</title>
<style>
  body { font-family: Arial, sans-serif; }
  .header { background: #6a11cb; color: white; padding: 20px; }
  .content { padding: 20px; }
  .footer { background: #333; color: white; padding: 10px; }
</style>
</head>
<body>
  <div class="header">
    <h1>Welcome to My Website!</h1>
  </div>
  <div class="content">
    <h2>About Me</h2>
    <p>I'm a 5th grader who loves coding!</p>
    <h3>My Favorite Things</h3>
    <ul>
      <li>Playing video games</li>
      <li>Reading books</li>
      <li>Creating websites</li>
    </ul>
  </div>
  <div class="footer">
    <p>Created by Me © 2023</p>
  </div>
</body>
</html>
