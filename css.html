What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages — to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

How CSS Works

CSS works by associating rules with HTML elements. These rules govern how the content of specified elements should be displayed. A CSS rule contains two parts: a selector and a declaration.

/* This is a CSS comment */
h1 {
  color: blue;
  font-size: 24px;
}

Ways to Add CSS

  • External CSS: Styles are defined in a separate .css file
  • Internal CSS: Styles are defined in the <style> section of an HTML document
  • Inline CSS: Styles are applied directly to HTML elements using the style attribute

CSS Syntax

A CSS rule consists of a selector and a declaration block:

selector {
  property: value;
}

CSS Declaration

Each declaration includes a CSS property name and a value, separated by a colon. Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

p {
  color: red;
  text-align: center;
}

This paragraph is styled with CSS - red and centered!

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Basic Selectors

Selector Example Description
Element p Selects all <p> elements
ID #header Selects the element with id="header"
Class .intro Selects all elements with class="intro"
Universal * Selects all elements
Grouping h1, p Selects all <h1> and <p> elements

Selector Examples

/* Element selector */
p { color: blue; }

/* ID selector */
#main { background: #f4f4f4; }

/* Class selector */
.highlight { background: yellow; }

/* Grouping selector */
h1, h2, h3 { font-family: Arial; }

CSS Colors

Colors in CSS can be specified in different ways:

Color Values

Method Example Description
Color Names red, blue, green Predefined color names
HEX #ff0000, #00ff00 Hexadecimal values
RGB rgb(255, 0, 0) Red, Green, Blue values
RGBA rgba(255, 0, 0, 0.5) RGB with transparency
HSL hsl(0, 100%, 50%) Hue, Saturation, Lightness

Color Examples

.color-example {

CSS Box Model

All HTML elements can be considered as boxes. In CSS, the term "box model" is used when talking about design and layout.

Box Model Components

  • Content: The content of the box, where text and images appear
  • Padding: Clears an area around the content (inside the border)
  • Border: A border that goes around the padding and content
  • Margin: Clears an area outside the border (outside the element)
.box {
  width: 300px;
  padding: 20px;
  border: 5px solid gray;
  margin: 15px;
}

This is the content of the box. The total width of this element is 300px + 40px (padding) + 10px (border) = 350px.

CSS Layout

CSS provides several layout techniques for positioning elements on a web page.

Display Property

The display property specifies the display behavior of an element.

/* Block elements take full width */
.block { display: block; }

/* Inline elements take only necessary width */
.inline { display: inline; }

/* Inline-block combines features of both */
.inline-block { display: inline-block; }

/* Hide an element */
.hidden { display: none; }

Position Property

The position property specifies the type of positioning method used for an element.

/* Static - default position */
.static { position: static; }

/* Relative - positioned relative to its normal position */
.relative { position: relative; }

/* Absolute - positioned relative to the nearest positioned ancestor */
.absolute { position: absolute; }

/* Fixed - positioned relative to the viewport */
.fixed { position: fixed; }

/* Sticky - toggles between relative and fixed */
.sticky { position: sticky; }

Responsive Design

Responsive web design makes your web page look good on all devices (desktops, tablets, and phones).

Viewport Meta Tag

The viewport is the user's visible area of a web page. It varies with the device.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Media queries allow you to apply CSS styles depending on a device's characteristics.

/* For devices with width 600px and smaller */
@media screen and (max-width: 600px) {
  body {
    background-color: lightblue;
  }
  .column {
    width: 100%;
  }
}

Responsive Units

  • %: Percentage - relative to the parent element
  • vw: Viewport width - 1vw = 1% of viewport width
  • vh: Viewport height - 1vh = 1% of viewport height
  • rem: Relative to root element font-size
  • em: Relative to parent element font-size

CSS Flexbox

Flexbox is a one-dimensional layout method for laying out items in rows or columns.

Flex Container Properties

.container {
  display: flex;
  flex-direction: row; /* row, row-reverse, column, column-reverse */
  justify-content: center; /* flex-start, flex-end, center, space-between, space-around */
  align-items: center; /* stretch, flex-start, flex-end, center, baseline */
  flex-wrap: wrap; /* nowrap, wrap, wrap-reverse */
}

CSS Grid

CSS Grid Layout is a two-dimensional layout system for the web.

Grid Container Properties

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* Three equal columns */
  grid-template-rows: 100px 200px; /* Two rows */
  gap: 10px; /* Space between grid items */
}

CSS Transitions & Animations

CSS transitions and animations allow you to change property values smoothly over a given duration.

Transitions

CSS transitions allows you to change property values smoothly, over a given duration.

.box {
  width: 100px;
  height: 100px;
  background: red;
  transition: width 2s, height 2s, background 2s;
}

.box:hover {
  width: 200px;
  height: 200px;
  background: blue;
}
Hover over me!

Animations

CSS animations make it possible to animate transitions from one CSS style configuration to another.

/* Define the animation */
@keyframes example {
  0% { background-color: red; }
  50% { background-color: yellow; }
  100% { background-color: red; }
}

/* Apply the animation */
.animated-box {
  animation-name: example;
  animation-duration: 4s;
  animation-iteration-count: infinite;
}
