What is JavaScript?

JavaScript is a programming language that adds interactivity to your website. It works in web browsers to create dynamic content, control multimedia, animate images, and much more.

Your First JavaScript

// This will show a popup message
alert('Hello, World!');

Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using let, const, or var.

// Declaring variables
let name = "John";
const age = 25;
var isStudent = true;

// Using variables
console.log(name); // Output: John
console.log(age); // Output: 25

Functions

Functions are blocks of code designed to perform a particular task. They are executed when "called" or "invoked".

// Function declaration
function greet(name) {
  return "Hello, " + name + "!";
}

// Calling the function
let message = greet("Sarah");
console.log(message); // Output: Hello, Sarah!

Events

Events are actions that happen in the browser, like clicks, mouse movements, or key presses. JavaScript can "listen" for these events and respond to them.

// Adding a click event listener
document.getElementById("myButton").addEventListener("click", function() {
  alert("Button was clicked!");
});

DOM Manipulation

The Document Object Model (DOM) is a programming interface for HTML. JavaScript can manipulate the DOM to change content, styles, and structure of a webpage.

// Changing HTML content
document.getElementById("demo").innerHTML = "New content!";

// Changing CSS styles
document.getElementById("demo").style.color = "red";

// Creating new elements
let newElement = document.createElement("p");
newElement.innerHTML = "This is a new paragraph";
document.body.appendChild(newElement);

Conditional Statements

Conditional statements are used to perform different actions based on different conditions.

let time = 14;

if (time < 12) {
  console.log("Good morning!");
} else if (time < 18) {
  console.log("Good afternoon!");
} else {
  console.log("Good evening!");
}

Loops

Loops can execute a block of code multiple times. They are handy when you want to run the same code over and over again, each time with a different value.

// For loop - repeats a block of code a number of times
for (let i = 0; i < 5; i++) {
  console.log("Number: " + i);
}

// While loop - repeats while a condition is true
let count = 0;
while (count < 3) {
  console.log("Count: " + count);
  count++;
}

Arrays

Arrays are used to store multiple values in a single variable. They are especially useful when you want to store a list of items.

// Creating an array
let fruits = ["Apple", "Banana", "Orange"];

// Accessing array elements
console.log(fruits[0]); // Output: Apple

// Adding to an array
fruits.push("Grapes");

// Array length
console.log(fruits.length); // Output: 4

Objects

Objects are collections of key-value pairs. They are used to store multiple properties that belong to the same entity.

// Creating an object
let person = {
  firstName: "John",
  lastName: "Doe",
  age: 30,
  isStudent: false,
  fullName: function() {
    return this.firstName + " " + this.lastName;
  }
};

// Accessing object properties
console.log(person.firstName); // Output: John
console.log(person.fullName()); // Output: John Doe
