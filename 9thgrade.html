Classes & Objects Basics

What is Object-Oriented Programming?

OOP is a way of programming that organizes code into "objects" that represent real-world things. Instead of just functions and variables, we create blueprints (classes) for objects.

Key OOP Concepts:

  • Class: A blueprint for creating objects
  • Object: An instance of a class (actual thing)
  • Attributes: Data stored in objects (variables)
  • Methods: Functions that belong to objects

Real-World Analogy

Think of a class as a blueprint for a house. The blueprint defines what a house should have (rooms, doors, windows). An object is an actual house built from that blueprint. You can build many houses from the same blueprint!

Creating Your First Class

# Define a Student class (blueprint)
class Student:
    # Special method called when object is created
    def __init__(self, name, grade, subjects):
        self.name = name # Attribute
        self.grade = grade # Attribute
        self.subjects = subjects # Attribute
        self.attendance = 100 # Default attribute

# Create objects (actual students)
student1 = Student("Anya Sharma", 9, ["Math", "Science"])
student2 = Student("Rohan Patel", 9, ["English", "Art"])

# Access object attributes
print("Student 1:", student1.name)
print("Student 2 grade:", student2.grade)
print("Anya's subjects:", student1.subjects)
Student 1: Anya Sharma
Student 2 grade: 9
Anya's subjects: ['Math', 'Science']

Methods & Attributes

Adding Behavior to Objects

Methods are functions that belong to objects. They define what objects can do, while attributes define what objects have.

Creating Methods

class Student:
    def __init__(self, name, grade):
        self.name = name
        self.grade = grade
        self.marks = {}

    # Method to add marks
    def add_mark(self, subject, mark):
        self.marks[subject] = mark
        print(f"✅ Added {mark} in {subject} for {self.name}")

    # Method to calculate average
    def calculate_average(self):
        if not self.marks:
            return 0
        total = sum(self.marks.values())
        return total / len(self.marks)

    # Method to display info
    def display_info(self):
        print(f"\n📊 Student: {self.name}")
        print(f"Grade: {self.grade}")
        print(f"Marks: {self.marks}")
        print(f"Average: {self.calculate_average():.1f}")

# Using the class
student = Student("Priya Kumar", 9)
student.add_mark("Math", 85)
student.add_mark("Science", 92)
student.add_mark("English", 78)
student.display_info()
✅ Added 85 in Math for Priya Kumar
✅ Added 92 in Science for Priya Kumar
✅ Added 78 in English for Priya Kumar

📊 Student: Priya Kumar
Grade: 9
Marks: {'Math': 85, 'Science': 92, 'English': 78}
Average: 85.0

Inheritance & Reusability

What is Inheritance?

Inheritance allows you to create new classes based on existing classes. The new class (child) inherits all attributes and methods from the existing class (parent), and can add its own features.

Real-World Example

Think of a Vehicle class as a parent with basic features (wheels, engine). Car and Bicycle classes can inherit from Vehicle and add their own special features (Car has doors, Bicycle has pedals).

Basic Inheritance

# Parent class
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def introduce(self):
        print(f"Hello, I'm {self.name} and I'm {self.age} years old.")

# Child class inheriting from Person
class Student(Person):
    def __init__(self, name, age, grade):
        # Call parent's __init__ method
        super().__init__(name, age)
        self.grade = grade
        self.subjects = []

    def add_subject(self, subject):
        self.subjects.append(subject)
        print(f"📚 {self.name} is now studying {subject}")

# Using the classes
person = Person("Mr. Gupta", 45)
student = Student("Anya", 14, 9)

person.introduce()
student.introduce() # Inherited from Person!
student.add_subject("Mathematics")
print(f"{student.name} is in grade {student.grade}")
Hello, I'm Mr. Gupta and I'm 45 years old.
Hello, I'm Anya and I'm 14 years old.
📚 Anya is now studying Mathematics
Anya is in grade 9

File Handling Basics

Working with Files

File handling allows your programs to read from and write to files, making data persistent between program runs. This is essential for saving user data, configurations, and program results.

Reading and Writing Files

# Writing to a file
# Create and write to a file
with open("student_records.txt", "w") as file:
    file.write("Student Records\n")
    file.write("===============\n")
    file.write("Anya Sharma, Grade 9, Mathematics: 85\n")
    file.write("Rohan Patel, Grade 9, Science: 92\n")
    file.write("Priya Kumar, Grade 9, English: 78\n")

print("✅ Student records saved to file!")

# Reading from a file
print("\n📖 Reading from file:")
with open("student_records.txt", "r") as file:
    content = file.read()
    print(content)

# Reading line by line
print("\n📝 File content line by line:")
with open("student_records.txt", "r") as file:
    for line in file:
        print(f"Line: {line.strip()}")

Final Project: Library Management System

Create a complete library system using object-oriented programming principles.

# Library Management System - Complete Project
class Book:
    def __init__(self, title, author, isbn):
        self.title = title
        self.author = author
        self.isbn = isbn
        self.is_available = True

    def borrow(self):
        if self.is_available:
            self.is_available = False
            return True
        return False

    def return_book(self):
        self.is_available = True

    def __str__(self):
        status = "Available" if self.is_available else "Borrowed"
        return f"'{self.title}' by {self.author} - {status}"

class Library:
    def __init__(self):
        self.books = []

    def add_book(self, book):
        self.books.append(book)
        print(f"✅ Added: {book}")

    def display_books(self):
        if not self.books:
            print("📚 No books in library.")
            return
        print("\n📚 Library Collection:")
        for i, book in enumerate(self.books, 1):
            print(f" {i}. {book}")

    def find_book(self, title):
        for book in self.books:
            if book.title.lower() == title.lower():
                return book
        return None

# Main program
library = Library()

# Add some books
library.add_book(Book("Python Programming", "Alex Coder", "12345"))
library.add_book(Book("Mathematics Basics", "Dr. Math", "12346"))
library.add_book(Book("Science Wonders", "Prof. Science", "12347"))

# Display all books
library.display_books()

# Borrow a book
book = library.find_book("python programming")
if book:
    if book.borrow():
        print(f"\n✅ Successfully borrowed: {book.title}")
    else:
        print(f"\n❌ {book.title} is already borrowed")

# Display books again to see changes
library.display_books()
