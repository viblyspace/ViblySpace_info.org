What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. It's widely used in web development, data science, artificial intelligence, and more.

Variables

Variables are containers for storing data values. In Python, you don't need to declare variables with any particular type.

# Declaring variables
name = "John"
age = 25
is_student = True

# Using variables
print(name) # Output: John
print(age) # Output: 25

Functions

Functions are blocks of code that only run when they are called. You can pass data, known as parameters, into a function.

# Function definition
def greet(name):
  return "Hello, " + name + "!"

# Calling the function
message = greet("Sarah")
print(message) # Output: Hello, Sarah!

Conditional Statements

Conditional statements are used to perform different actions based on different conditions.

time = 14

if time < 12:
  print("Good morning!")
elif time < 18:
  print("Good afternoon!")
else:
  print("Good evening!")

Loops

Loops are used to execute a block of code multiple times. Python has two types of loops: for loops and while loops.

# For loop - iterates over a sequence
for i in range(5):
  print("Number:", i)

# While loop - repeats while condition is true
count = 0
while count < 3:
  print("Count:", count)
  count += 1

Lists

Lists are used to store multiple items in a single variable. They are one of 4 built-in data types in Python used to store collections of data.

# Creating a list
fruits = ["Apple", "Banana", "Orange"]

# Accessing list elements
print(fruits[0]) # Output: Apple

# Adding to a list
fruits.append("Grapes")

# List length
print(len(fruits)) # Output: 4

Dictionaries

Dictionaries are used to store data values in key:value pairs. They are unordered, changeable and do not allow duplicates.

# Creating a dictionary
person = {
  "first_name": "John",
  "last_name": "Doe",
  "age": 30,
  "is_student": False
}

# Accessing dictionary values
print(person["first_name"]) # Output: John
print(person.get("age")) # Output: 30

Modules

Modules are files containing Python code that can be imported and used in other Python programs. Python has a rich collection of built-in modules.

# Importing a module
import math

# Using module functions
print(math.sqrt(16)) # Output: 4.0
print(math.pi) # Output: 3.141592653589793

# Importing specific functions
from datetime import datetime
print(datetime.now()) # Output: current date and time
