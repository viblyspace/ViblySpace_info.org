What is Web Design?

Web design is the art of creating websites. It's like building a digital house where people can visit and find information.

When you visit any website like YouTube, Wikipedia, or your school's website, you're looking at web design in action!

Think of it like building a house:

HTML is the structure - the walls, floors, and rooms

CSS is the decoration - the paint, furniture, and decorations

Together they create a complete, beautiful house that people enjoy visiting!

HTML - The Building Blocks

HTML stands for HyperText Markup Language. It's the foundation of every website.

What are HTML Tags?

HTML uses special codes called tags to create different parts of a webpage. Tags are like labels that tell the browser what each piece of content is.

Tags usually come in pairs - an opening tag and a closing tag:

<tagname>Content goes here</tagname>

The closing tag has a forward slash / before the tag name.

Basic HTML Structure

Every HTML document has the same basic structure. Think of it as a recipe that always starts the same way:

<!DOCTYPE html> <html> <head> <title>My Webpage Title</title> </head> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> </body> </html>

Remember: The <head> contains information about the page (like the title), and the <body> contains all the visible content.

Most Important HTML Tags

Headings: <h1> to <h6> - Create headings of different sizes

Paragraphs: <p> - Create paragraphs of text

Links: <a href="url">Link Text</a> - Create clickable links

Images: <img src="image.jpg" alt="description"> - Show pictures

Lists: <ul>, <ol>, <li> - Create bulleted or numbered lists

Division: <div> - Create containers to group elements

CSS - Making Websites Beautiful

CSS stands for Cascading Style Sheets. It's what makes websites look pretty and organized.

If HTML is a plain black and white coloring book, CSS is your box of crayons that brings everything to life with color!

How CSS Works

CSS works by selecting HTML elements and telling them how to look. It's like giving instructions:

"Make all headings blue" or "Give paragraphs a yellow background"

CSS Syntax

CSS uses a simple pattern:

selector { property: value; property: value; }

Example:

h1 { color: blue; font-size: 24px; text-align: center; }

This CSS makes all h1 headings blue, 24 pixels big, and centered on the page.

Common CSS Properties

Colors and Text:

  • color - Changes text color
  • background-color - Changes background color
  • font-family - Changes font style
  • font-size - Changes text size
  • text-align - Aligns text (left, center, right)

Spacing and Layout:

  • width and height - Set element size
  • margin - Space outside an element
  • padding - Space inside an element
  • border - Creates borders

CSS Selectors

Selectors tell CSS which HTML elements to style:

Element Selector: Styles all elements of that type

p { color: green; }

Class Selector: Styles elements with a specific class (use . before class name)

.special { font-weight: bold; }

ID Selector: Styles one specific element (use # before ID name)

#main-title { font-size: 2rem; }

Adding Images and Links

Images Make Websites Interesting

Images help tell stories and make your website fun to look at. You add images with the <img> tag.

<img src="cat.jpg" alt="A cute cat playing with yarn">

Important: Always include alt text that describes the image. This helps people who can't see the image understand what it shows.

Links Connect Everything

Links let visitors move between pages. You create links with the <a> tag.

<a href="https://www.google.com">Visit Google</a>

The text between the tags is what people click on. The href tells the browser where to go when clicked.

Creating Lists and Tables

Lists Organize Information

Lists help present information in a clear, easy-to-read way.

Ordered Lists (numbered):

<ol> <li>First item</li> <li>Second item</li> </ol>

Unordered Lists (bulleted):

<ul> <li>Apples</li> <li>Bananas</li> </ul>

Tables for Complex Information

Tables organize information into rows and columns, perfect for schedules or comparisons.

<table> <tr> <th>Day</th> <th>Activity</th> </tr> <tr> <td>Monday</td> <td>Soccer practice</td> </tr> </table>

Colors and Fonts

Colors Express Feelings

Different colors create different moods:

Blue - Calm and trustworthy

Red - Exciting and urgent

Green - Natural and healthy

Yellow - Happy and energetic

Purple - Creative and royal

CSS Color Systems

You can specify colors in different ways:

Color Names: red, blue, green (140+ options)

Hex Codes: #FF0000 (red), #00FF00 (green)

RGB: rgb(255, 0, 0) (red), rgb(0, 255, 0) (green)

Fonts Give Personality

Fonts are the style of your text. CSS lets you control:

font-family - Which font to use

font-size - How big the text is

font-weight - How thick the text is (bold, normal)

font-style - Normal or italic

Design Tip: Use only 2-3 fonts on a webpage. Too many fonts can look messy and confusing.

Putting Everything Together

Website Structure

Most websites have these main parts:

Header - Top section with logo and navigation

Main Content - Central area with your information

Footer - Bottom section with contact info and links

Web Design Best Practices

Keep it simple - Don't overcrowd with too much content

Be consistent - Use the same styles throughout

Make it readable - Good contrast between text and background

Navigation should be clear - Easy to move around your site

Check your links - Make sure all links work
