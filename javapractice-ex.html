Student Management System

Create a comprehensive student management system to practice Java OOP concepts and data handling.

Learning Objectives:

  • Practice Object-Oriented Programming principles
  • Understand class design and relationships
  • Implement CRUD operations (Create, Read, Update, Delete)
  • Work with collections and data structures
  • Practice encapsulation and data validation

Instructions:

Build a student management system that can:

  • Add new students with details (ID, name, grade, course)
  • View all students in a list/table
  • Update student information
  • Delete students from the system
  • Search/filter students by various criteria

public class Student {

  private String grade;
  private String course;

  public Student(String id, String name, String grade, String course) {

    this.name = name;
    this.grade = grade;
    this.course = course;
  }

  // Getters and setters
  public String getId() { return id; }
  public void setId(String id) { this.id = id; }
  // ... other getters and setters
}

public class StudentManagementSystem {
  private List<Student> students = new ArrayList<>();

  public void addStudent(Student student) {
    // Add student to the list
    students.add(student);
  }

  public void updateStudent(String id, Student updatedStudent) {
    // Find and update student
    for (Student student : students) {
      if (student.getId().equals(id)) {
        student.setName(updatedStudent.getName());
        // ... update other fields
        break;
      }
    }
  }

  public void deleteStudent(String id) {
    // Remove student from the list
    students.removeIf(student -> student.getId().equals(id));
  }
}

// Example: Java implementation structure
public class BankAccount {
  private String accountNumber;
  private String accountHolder;
  private String accountType;
  private double balance;
  private List<Transaction> transactions = new ArrayList<>();

  public BankAccount(String accountNumber, String accountHolder, String accountType) {
    this.accountNumber = accountNumber;
    this.accountHolder = accountHolder;
    this.accountType = accountType;
    this.balance = 0.0;
  }

  public void deposit(double amount, String description) {
    if (amount > 0) {
      balance += amount;
      Transaction transaction = new Transaction("DEPOSIT", amount, description);
      transactions.add(transaction);
    }
  }

  public void withdraw(double amount, String description) {
    if (amount > 0 && amount <= balance) {
      balance -= amount;
      Transaction transaction = new Transaction("WITHDRAWAL", amount, description);
      transactions.add(transaction);
    } else {
      // Throw exception or handle insufficient funds
    }
  }

  // Getters for account information
  public double getBalance() { return balance; }
  public List<Transaction> getTransactionHistory() { return transactions; }
}

Library Management System

Create a library management system to practice Java collections, search operations, and resource tracking.

Learning Objectives:

  • Practice working with Java Collections Framework
  • Understand resource management and availability tracking
  • Implement search and filter operations
  • Practice date handling for due dates and returns
  • Work with multiple entity relationships (books, members, loans)

Instructions:

Build a library management system that can:

  • Add books to the library catalog
  • Search for books by title, author, or ISBN
  • Check out books to members
  • Return books and update availability
  • Track due dates and calculate fines

// Example: Java implementation structure
public class Book {
  private String isbn;
  private String title;
  private String author;
  private String category;
  private boolean isAvailable;
  private String borrowedBy;
  private LocalDate dueDate;

  public Book(String isbn, String title, String author, String category) {
    this.isbn = isbn;
    this.title = title;
    this.author = author;
    this.category = category;
    this.isAvailable = true;
  }

  public void checkOut(String memberId) {
    if (isAvailable) {
      this.isAvailable = false;
      this.borrowedBy = memberId;
      this.dueDate = LocalDate.now().plusWeeks(2); // 2 weeks loan period
    }
  }

  public void returnBook() {
    this.isAvailable = true;
    this.borrowedBy = null;
    this.dueDate = null;
  }

  // Getters and setters
  public boolean isAvailable() { return isAvailable; }
  // ... other getters and setters
}

public class Library {
  private List<Book> books = new ArrayList<>();

  public void addBook(Book book) {
    books.add(book);
  }

  public List<Book> searchBooks(String query) {
    // Search books by title, author, or ISBN
    return books.stream()
      .filter(book -> book.getTitle().toLowerCase().contains(query.toLowerCase()) ||
        book.getAuthor().toLowerCase().contains(query.toLowerCase()) ||
        book.getIsbn().equals(query))
      .collect(Collectors.toList());
  }
}
